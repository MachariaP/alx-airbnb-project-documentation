
# User Stories

## Overview
User stories derived from the use case diagram for the Airbnb Clone backend.

## User Stories
1.  As a Guest, I want to register an account so that I can book properties and access personalized features.
2.  As a Host, I want to create a property listing so that guests can book my property.
3.  As a Guest, I want to search properties by location and amenities so that I can find a suitable stay.
4.  As a Guest, I want to process a payment securely so that I can confirm my booking.
5.   As an Admin, I want to manage user accounts so that I can ensure platform integrity.
6.  As a Guest, I want to receive booking confirmation notifications so that I can plan my trip.
